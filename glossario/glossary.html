Estrutura HTML

!DOCTYPE html
Define o tipo de documento como HTML5
html
Define um documento HTML
head
Container para metadados, título da página, links
body
Corpo ou conteúdo de um documento
header
Representa um grupo de suporte introdutório ou navegacional
main
Define o conteúdo principal da página
aside
Representa uma seção de uma página com conteúdo vagamente relacionado ao seu entorno. Ex: informações de perfil, links de blogs relacionados, avisos, explicações.
footer
Representa o rodapé da página, de um artigo, seção ou tabela. Contém informações de direitos autorais, links para documentos relacionados ou informações sobre o autor(es) da página
